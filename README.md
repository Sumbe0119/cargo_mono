# Mono repo setup
    - main package.json yarn2 workspace setup.
    - ESLint tsconfig setup.

# Development Guide
    - yarn # for installing dependencies
    # following commands added.
    - yarn lint # for linting projects.
    - yarn build # for building and bundling projects for production.
    - yarn cov # for integration-test and unit-test coverages using Jest.
    - you can start projects by switching specific project's directory and run ocuppied commands either yarn dev or yarn start
    - run `yarn wsi` command to see existing mono-repo list
